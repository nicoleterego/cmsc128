To do some changes to a file

> git add -A # adds all changed files to index
> git commit -m "message" # commits all files in an index
> git push <remote> <branch>

Branches

> git branch #shows us available branches and active branch
> git checkout <branch name> #switch to branch

git checkout -b <new branchname>

Merging of branch to master

Source branch --> target branch

Deleting git branch

git branch -d newfeature

Remote Repositories

> git remote add origin # add the specified url as a remote repository and name it 'origin'

> git push <remote> <branch> # push the local commit to the remote called origin, specifically, the master branch
